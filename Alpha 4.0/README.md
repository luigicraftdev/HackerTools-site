# ⚡ Hacker Tools

Hacker Tools is a futuristic cyber-style web application inspired by hacking terminals, cybersecurity dashboards, and sci-fi interfaces.

Built with Flask, HTML, and CSS, the project combines a modern hacker aesthetic with educational explanations about hash cracking and password security.

The interface includes glowing green terminal effects, animated grid overlays, neon typography, and a responsive Orbitron-based design to recreate the feeling of a real cyberpunk operating system.

---

# 🚀 Features

- 🔓 Hash Cracker interface
- 🧠 Hash cracking explanation system
- 🔐 Password difficulty explanation
- 💚 Cyberpunk terminal UI
- ⚡ Responsive design
- 🌐 Flask backend
- 🟩 Animated grid effects
- 🎬 Futuristic hacker aesthetic
